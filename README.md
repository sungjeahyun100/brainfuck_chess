# 덱체스 / Deck Chess

이 게임은 **덱빌딩 변형체스**입니다. 제한된 점수 안에서 자신만의 기물 덱을 구성하고, 시작 기물과 포켓 기물을 활용해 상대 King을 포획하는 것을 목표로 합니다. "어려운 룰을 최대한 많이 넣은 체스"가 아니라, **나만의 덱을 짜서 플레이하는 전략 체스**를 지향합니다.

> 개발 초기에는 "Brainfuck Chess"라는 이름으로 불렸습니다.

## 핵심 컨셉 한눈에 보기

1. **덱을 짜는 체스**: 게임을 시작하기 전, 각 플레이어는 자신만의 기물 조합으로 덱을 구성합니다.
2. **시작 기물 / 포켓 기물**: 덱은 게임 시작 시 보드에 미리 놓이는 **시작 기물**과, 게임 중 보드에 놓을 수 있는 **포켓 기물**로 나뉩니다.
3. **덱 제한 점수 안에서 구성**: 기물마다 점수가 있고, 덱 전체 점수는 보드 크기에 따른 제한 점수를 넘을 수 없습니다.
4. **포켓 기물 놓기**: 포켓 기물은 게임 도중 자신의 시작 진영이나 자신의 기물이 위협하는 칸에 내려놓을 수 있습니다.
5. **승리 조건**: 체크/체크메이트 없이, 상대 King을 실제로 **포획**하면 즉시 승리합니다.
6. **단일 행동 턴**: 한 턴에는 기물 이동 1회 또는 포켓 기물 놓기 1회 중 하나만 할 수 있습니다.

## 사용된 아이디어들
1. 포켓 기물 놓기(아이디어 제공:바둑): 포켓에 있는 기물을 보드 위에 내려놓아가며 플레이할 수 있습니다.
2. 더 큰 보드 사이즈(아이디어 제공:슥슥이 유튜브의 XXL-체스 쇼츠): 8*8이외에도 더 큰 보드에서 플레이할 수 있습니다.
3. 다양한 변형 기물등(아이디어 및 기술력 제공:smgk님의 Chessembly): 기본 체스 이외에도 다양한 변형 기물들을 사용하실 수 있습니다.

# 룰

## 보드

- 보드는 정사각형(`n x n`)이며, 최소 크기는 `8x8`입니다.
- 플레이어 선택에 따라 `9x9`, `10x10` 등 더 큰 보드에서 플레이할 수 있습니다.

## 덱 구성

게임 시작 전 각 플레이어는 자신의 덱을 구성합니다. 덱은 두 영역으로 나뉩니다.

### 시작 기물
- 게임 시작 시 자신의 기본 진영에 미리 놓고 시작하는 기물입니다.
- White 기본 진영: `rank 0`, `rank 1` / Black 기본 진영: `rank n-2`, `rank n-1`
- 각 플레이어는 반드시 **King 1개**를 기본 진영에 배치해야 합니다.
- **King은 필수이며, 포켓에 넣을 수 없고, 복수로 가질 수 없으며, 덱 점수 계산에서 제외됩니다.**

### 포켓 기물
- 게임 시작 시 보드 위에 없고, 게임 중 보드에 **놓을(포켓 기물 놓기)** 수 있는 기물입니다.
- 포켓 기물도 덱 제한 점수에 포함됩니다.
- King은 포켓에 넣을 수 없습니다.

## 덱 제한 점수

덱 제한 점수는 보드 크기에 따라 달라집니다.

```
scoreLimit = n × n - 25
```

| 보드 크기 | 덱 제한 점수 |
|---------|---------|
| 8×8     | 39점    |
| 9×9     | 56점    |
| 10×10   | 75점    |

기본 기물 점수:

| 기물   | 점수 |
|--------|------|
| Pawn   | 1    |
| Knight | 3    |
| Bishop | 3    |
| Rook   | 5    |
| Queen  | 9    |
| King   | 제외 |

덱 점수가 제한을 넘으면 초과한 만큼 점수를 줄여야 하며(예: `점수가 4점 초과되었습니다`), King은 항상 점수 계산에서 제외됩니다.

## 포켓 기물 놓기 규칙

포켓 기물은 다음 중 하나에 해당하는 **빈 칸**에 놓을 수 있습니다.

1. 자신의 기본 진영 칸
2. 현재 자신의 기물들이 공격하거나 위협하는 칸(공격 범위)

```
놓을 수 있는 칸 = 기본 진영 칸 ∪ 자신의 공격 범위
```

- 이미 기물이 있는 칸에는 놓을 수 없습니다.
- King은 포켓 기물로 놓을 수 없습니다.

## 프리셋 덱

덱을 처음부터 짜기 어렵다면, 덱 빌더 화면에서 바로 고를 수 있는 프리셋 덱을 제공합니다.

| 프리셋 | 컨셉 |
|--------|------|
| 기본 체스 덱 | 익숙한 기물 중심의 표준 배치 |
| 물량 덱 | 낮은 점수 기물(Knight, Pawn)을 많이 사용 |
| 포켓 덱 | 시작 기물은 최소화하고 포켓 기물을 많이 사용 |
| 기동 덱 | Knight/Bishop과 변형 기물 중심의 기동력 강화 |
| 고화력 덱 | Queen/Rook 같은 고점수 기물 중심 |

프리셋을 선택한 뒤에도 자유롭게 기물을 추가/제거해 자신만의 덱으로 다듬을 수 있습니다.

## 승리 조건

- **체크와 체크메이트는 없습니다.**
- 상대 King을 실제로 **포획**하면 즉시 승리합니다.
- King은 공격받는 칸으로도 이동할 수 있습니다.
- King이 포획되는 순간 게임은 즉시 종료됩니다.

## 턴 진행 방식

한 턴에는 **이동 모드** 또는 **포켓 기물 놓기 모드** 중 하나만 선택할 수 있습니다.

- **이동 턴**: 자신의 보드 위 기물 1개를 1회 이동합니다.
- **포켓 기물 놓기 턴**: 포켓 기물 1개를 보드에 1회 놓습니다.
- 한 턴에 이동과 포켓 기물 놓기를 함께 할 수 없습니다.
- 이동 또는 포켓 기물 놓기가 완료되면 자동으로 상대 턴으로 넘어갑니다.
- **패스는 불가**합니다. 반드시 기물 이동 1회 또는 포켓 기물 놓기 1회를 해야 합니다.

## 기물 이동 규칙

기본 체스 기물의 움직임을 따르되 아래 예외가 있습니다.

### Pawn
- White는 rank 증가 방향, Black은 rank 감소 방향으로 전진합니다.
- 이동 범위와 공격 범위가 분리됩니다. (공격은 대각선 전방 1칸)
- 자기 기본 진영의 두 번째 랭크에 있을 때만 2칸 전진할 수 있습니다.

### 캐슬링(Castling)

캐슬링은 정식 룰이며, 다음 조건을 모두 만족해야 합니다.

- King과 캐슬링에 참여하는 Rook이 모두 아직 한 번도 이동하지 않았어야 합니다.
- King과 Rook 사이의 모든 칸이 비어 있어야 합니다.
- King이 현재 상대 기물의 공격 범위에 있으면 캐슬링할 수 없습니다.
- King이 지나가는 칸 또는 도착하는 칸이 상대 기물의 공격 범위에 있으면 캐슬링할 수 없습니다.
- 일반 이동에서는 King이 공격받는 칸으로 이동할 수 있지만, 캐슬링에 한해서만 위 공격 범위 제한을 적용합니다.

### 앙파상(En Passant)

앙파상은 정식 룰이며, 다음과 같이 처리합니다.

- Pawn이 자신의 기본 진영 두 번째 랭크에서 처음으로 2칸 전진하면 앙파상 기회가 생깁니다.
- 상대 Pawn은 방금 전진한 Pawn이 지나간 칸으로 대각선 이동하여 포획할 수 있습니다.
- 실제 포획 대상은 도착 칸의 기물이 아니라 방금 2칸 전진한 Pawn입니다.
- 앙파상 권리는 상대가 즉시 응답할 수 있는 기회에만 유효합니다.
- 상대가 다른 행동을 선택하거나 턴 또는 모드 진행으로 기회를 넘기면 권리는 소멸합니다.
