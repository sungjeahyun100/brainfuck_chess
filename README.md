# Brainfuck Chess

이 게임은 **가장 어려운 변형체스** 를 목표로 하는 변형체스입니다. 따라서 제작자가 지금까지 보았던 가장 어려워 보이는 룰들을 비빔밥마냥 전부 때려넣어 만들었습니다.

## 사용된 아이디어들
1. 한 턴에 여러 기물을 움직임(아이디어 제공:잉체스 유튜브의 콜로니-체스):모든 기물을 한 턴에 전부 움직여볼 수 있으며, 무한 턴 넘기기를 방지하기 위해 적어도 하나의 기물은 움직여야 합니다.
2. 착수(아이디어 제공:바둑): 기물을 보드 위에 **착수**해가며 플레이할 수 있습니다.
3. 더 큰 보드 사이즈(아이디어 제공:슥슥이 유튜브의 XXL-체스 쇼츠): 8*8이외에도 더 큰 보드에서 플레이할 수 있습니다.
4. 다양한 변형 기물등(아이디어 및 기술력 제공:smgk님의 Chessembly): 기본 체스 이외에도 다양한 변형 기물들을 사용하실 수 있습니다.

# 룰

## 보드

- 보드는 정사각형(`n x n`)이며, 최소 크기는 `8x8`입니다.
- 플레이어 선택에 따라 `9x9`, `10x10` 등 더 큰 보드에서 플레이할 수 있습니다.

## 덱 구성

게임 시작 전 각 플레이어는 자신의 덱을 구성합니다. 덱은 두 영역으로 나뉩니다.

### 기본 진영 배치 기물
- 게임 시작 시 자신의 기본 진영에 미리 놓고 시작하는 기물입니다.
- White 기본 진영: `rank 0`, `rank 1` / Black 기본 진영: `rank n-2`, `rank n-1`
- 각 플레이어는 반드시 **King 1개**를 기본 진영에 배치해야 합니다.
- King은 포켓에 넣을 수 없으며, 복수로 가질 수 없습니다.
- King은 덱 점수 계산에서 제외됩니다.

### 포켓 기물
- 게임 시작 시 보드 위에 없고, 게임 중 **착수**할 수 있는 기물입니다.
- 포켓 기물도 덱 점수 상한에 포함됩니다.
- King은 포켓에 넣을 수 없습니다.

## 덱 점수 상한

덱 점수 상한은 보드 크기에 따라 달라집니다.

```
scoreLimit = n × n - 25
```

| 보드 크기 | 점수 상한 |
|---------|---------|
| 8×8     | 39점    |
| 9×9     | 56점    |
| 10×10   | 75점    |

기본 기물 점수:

| 기물   | 점수 |
|--------|------|
| Pawn   | 1    |
| Knight | 3    |
| Bishop | 3    |
| Rook   | 5    |
| Queen  | 9    |
| King   | 제외 |

## 승리 조건

- **체크와 체크메이트는 없습니다.**
- 상대 King을 실제로 **포획**하면 즉시 승리합니다.
- King은 공격받는 칸으로도 이동할 수 있습니다.
- King이 포획되는 순간 게임은 즉시 종료됩니다.

## 턴 진행 방식

한 턴에는 **이동 모드** 또는 **착수 모드** 중 하나만 선택할 수 있습니다.

- **이동 턴**: 여러 기물을 이동할 수 있습니다.
- **착수 턴**: 포켓 기물 1개만 착수할 수 있습니다.
- **패스는 불가**합니다. 반드시 최소 하나의 행동을 해야 합니다.

## 이동 스택 시스템

- 매 턴 시작 시 현재 플레이어의 보드 위 기물들은 이동 스택 **1개**를 받습니다.
- 기물이 한 번 이동하면 해당 기물의 이동 스택 1개를 소비합니다.
- 이동 스택이 0인 기물은 그 턴에 더 이상 이동할 수 없습니다.
- 같은 기물은 한 턴에 최대 한 번만 이동할 수 있습니다.

## 착수 규칙

포켓 기물은 다음 중 하나에 해당하는 **빈 칸**에 착수할 수 있습니다.

1. 자신의 기본 진영 칸
2. 현재 자신의 기물들이 공격하거나 위협하는 칸

```
착수 가능 칸 = 기본 진영 칸 ∪ 자신의 Attack Map
```

- 이미 기물이 있는 칸에는 착수할 수 없습니다.
- King은 착수할 수 없습니다.

## 기물 이동 규칙

기본 체스 기물의 움직임을 따르되 아래 예외가 있습니다.

### Pawn
- White는 rank 증가 방향, Black은 rank 감소 방향으로 전진합니다.
- 이동 범위와 공격 범위가 분리됩니다. (공격은 대각선 전방 1칸)
- 자기 기본 진영의 두 번째 랭크에 있을 때만 2칸 전진할 수 있습니다.

### 캐슬링(Castling)

캐슬링은 정식 룰이며, 다음 조건을 모두 만족해야 합니다.

- King과 캐슬링에 참여하는 Rook이 모두 아직 한 번도 이동하지 않았어야 합니다.
- King과 Rook 사이의 모든 칸이 비어 있어야 합니다.
- King이 현재 상대 기물의 공격 범위에 있으면 캐슬링할 수 없습니다.
- King이 지나가는 칸 또는 도착하는 칸이 상대 기물의 공격 범위에 있으면 캐슬링할 수 없습니다.
- 일반 이동에서는 King이 공격받는 칸으로 이동할 수 있지만, 캐슬링에 한해서만 위 공격 범위 제한을 적용합니다.

### 앙파상(En Passant)

앙파상은 정식 룰이며, 다음과 같이 처리합니다.

- Pawn이 자신의 기본 진영 두 번째 랭크에서 처음으로 2칸 전진하면 앙파상 기회가 생깁니다.
- 상대 Pawn은 방금 전진한 Pawn이 지나간 칸으로 대각선 이동하여 포획할 수 있습니다.
- 실제 포획 대상은 도착 칸의 기물이 아니라 방금 2칸 전진한 Pawn입니다.
- 앙파상 권리는 상대가 즉시 응답할 수 있는 기회에만 유효합니다.
- 상대가 다른 행동을 선택하거나 턴 또는 모드 진행으로 기회를 넘기면 권리는 소멸합니다.
